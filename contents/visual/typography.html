Typography

Good typography is invisible. People should have the most seamless experience while they're using Firefox products.

Typefaces

In Firefox products we use the standard typeface for each platform.

Android

Roboto is the standard typeface on Android. Noto is the standard typeface for all languages on Chrome and Android for all languages not covered by Roboto.

iOS, macOS

San Francisco is the standard typeface on the Apples products.

Linux

Ubuntu is the typeface used in the most common Linux distro, Ubuntu.

Web products

FiraSans is the typeface used on the Firefox products on the web.

Despite Open Sans being use on some Mozilla web properties it is currently not recommended to be used in Firefox products.

Windows

Segoe UI is the typeface used on the Windows platform.

Emphasis

If two texts feature the same scale use a bolder font weight 500 to emphises the most important information.

Color

Choose text color based on purpose. Primary ( Grey 90 on light background, White on dark background) for titles and body text. Secondary for accompanying text that gives further information, details or features to the primary content such as caption text and subtitle of menus. Use the secondary color ( Grey 50 on light background, Grey 10 on dark background) for accompanying text that gives further information, details or features to the primary content such as caption text and subtitle of menus.

Titles and text

Display

Display style is recommeded to be used on in-content pages for delivering primary messages to users.

Title 10

Title style is used on top as a summary of the following content. A good pair of titles can improve visual readibility of a long content
