Grid

When all of your measurements follow the same rules, you automatically get a more consistent UI.

Baseline grid

The grid is based on a 4px square. This value controls the proportion, balance and vertical alignment of every Firefox components. All elements, including icons, aligned to this unit.

You can use Nudg.it to set keyboard increment to 4px in Sketch.

Spacing

When you design a component, elements are spaced following the grid. The same principle applies when you stack multiple elements within and between a component.
