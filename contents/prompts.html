Authentication Prompts

Use authentication prompts when websites require that users sign in to access any content from the server.

Modal Prompts

Use modal prompts when a user must interact with the dialog before further interactions with the application or website can occur. There are two types of modal prompts:

  • Tab modal, where the prompt appears anchored beneath the awesome bar.
  • Application modal, where the prompt appears over the awesome bar, anchored to the top chrome.

When more than prompt must be addressed, indicate the total number in the upper corner of the prompt and surface the next one in sequence as the user completes the previous one.
