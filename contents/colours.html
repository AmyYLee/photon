States

Firefox components usually have three different states: active, hover, and pressed. The active color is 500, the hover colour is one step darker which is 600 and the pressed colour is two steps darker which is 900.

Validation States

When you design a component that has validation use the following colours for success, warning or error. Validation state colours shloud only be used as background colour and not as text colour.
