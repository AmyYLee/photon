Link

Link is a text that users can click to quickly access websites.

Usage

Use text links as navigation to guide users to a new page or to anchor users to a section within a page. For calling an action that results in an immediate change, use buttons instead. See buttons

Types

Internal link

Internal links direct users to a page inside the Mozilla properties (e.g. SUMO, Pocket) or jump into a specific part on the same page.

External link

External links direct users to an external page which is not produced by Mozilla.
An Open In New icon should be paired with the external link.
