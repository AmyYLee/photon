The Scaling Ratio

The base for the Photon type scale is 1em. The ratio used to create the rest of the scale is 1.2. The ratio is multiplied by the base to produce a scale of numbers that is proportionally related. To provide more color to the scale, we skip one step every five.

Line-height

Ideal line-heights for standard copy have a ratio of 1:1.4 (typesize : line-height). For example, a type at 16px/1rem would have a line-height of 1.4rem/22px (16 x 1.4). The exception to this rule are headings, which need less spacing. The ideal ratio is 1:1.25.

Font Weight

Font weight helps to add emphasis and is used to differentiate content hierarchy. A bold weight will have more emphasis than a lighter weight font of the same size. However, a lighter weight text can rank hierarchically higher than a bold text if the lighter weight type size is significantly larger than the bold.

Line Length

The number of characters per line influences readability. Studies indicate the following guidelines for English:

  • Satisfactory
    45-75 characters (desktop) or 35-40 characters (mobile) per line is widely regarded as a satisfactory line length.
  • Optimal
    Around 66 characters (desktop) or 39 characters (mobile) is optimal.
  • Too long
    More than 90 characters (desktop) or 50 characters (mobile) is likely too long for continuous reading.
