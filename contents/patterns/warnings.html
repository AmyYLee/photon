Warnings

Components using warning states:

  • Message bars
  • Notification bars
  • Input field

Usage

Warnings provide short and relevant information about a specific situation. If possible, give to the user an action that will help them address the warning.

Placement

Warnings are usually place close to the context.

Style

Icon

Warnings text come taogether with an exclamation mark enclosed within a triangle.

Color

Yellow is the dedicated color for warning states. Warning text should be legible, with noticeable contrast against its background color.

Background: Yellow 50 #ffe900

Text color: Yellow 90 #3e2800

Icon color: Yellow 90 #3e2800
