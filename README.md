# Firefox UX Style Guide  [![Build Status](https://travis-ci.org/FirefoxUX/StyleGuide.svg?branch=master)](https://travis-ci.org/FirefoxUX/StyleGuide)

Requires:
* npm version: 3.5.3
* node version: 5.3.0

To run (with hot-reloading!):
```
$ npm install
$ npm run serve
```

To build:
```
$ npm install
$ npm run build
```

Pull requests welcome by anyone!
